# 🚀 Com pots contribuir?

##### Serveix com a guia per a les persones que volen ajudar, explicant des de com informar d'un error fins a com enviar canvis en el codi.

## 1. Informar d'errors (Bugs)
Si trobes algun problema, si us plau, obre un Issue a GitHub:

Fes servir un títol descriptiu.

Explica els passos per reproduir l'error.

## 2. Suggerir funcionalitats
Vols veure alguna cosa nova? Estem encantats d'escoltar-te! Obre un Issue amb l'etiqueta enhancement i explica per què creus que la funcionalitat seria útil.

## 3. Enviar canvis (Pull Requests)
Per contribuir amb codi, segueix aquest flux de treball:

1. Fes un Fork del repositori.
2. Crea una branca nova per a la teva millora.
3. Fes els teus canvis i assegura't que el codi segueix l'estil del projecte.
4. Fes un Commit clar i concís.
5. Fes un Push a la teva branca.
6. Obre una Pull Request (PR) explicant els canvis que has fet.

## 4. 🤝 Codi de Conducta
En participar en aquest projecte, acceptes complir el nostre Codi de Conducta, que fomenta un ambient respectuós i inclusiu per a tothom.
